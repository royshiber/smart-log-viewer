# מדריך הגדרת Gemini API Key

## הדרך הקלה (מומלצת)

**לחץ פעמיים על הקובץ:** `הגדר-API-Key.bat`

הסקריפט יפתח את הדפדפן ואת קובץ ההגדרות. עקוב אחרי ההודעות על המסך.

---

## הדרך הידנית

### שלב 1: קבלת המפתח
1. פתח בדפדפן: https://aistudio.google.com/apikey
2. התחבר עם Google
3. לחץ **Create API Key**
4. לחץ **Copy** להעתקת המפתח

### שלב 2: הדבקה בפרויקט
1. פתח את התיקייה: `תוכנת לוגים\server`
2. פתח את הקובץ `.env` (אם אין – צור קובץ חדש בשם `.env`)
3. כתוב בשורה הראשונה:
   ```
   GEMINI_API_KEY=הדבק_כאן_את_המפתח
   ```
4. שמור (Ctrl+S)

### שלב 3: הפעלה מחדש
1. סגור את הטרמינל שבו רץ הפרויקט
2. פתח טרמינל חדש
3. הרץ: `npm run dev`

---

## בדיקה
פתח את האפליקציה, טען קובץ BIN, לחץ "ניתוח עם Gemini" ושאל שאלה.
